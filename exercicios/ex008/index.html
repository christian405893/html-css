principais formatações

negrito / destaque

nessa frase temos um termo em negrito usando a tag <b>

nesta frase, tenho um termo em destaque usando a tag < strong > a tag (semântica).

itálico / ênfase

nesta frase, tenho um termo em itálico usando a tag (não tem semântica).

Nesta frase, temos um termo em ênfase usando a tag <em> (tem semântica)

para a atualidade use sempre <strong> para negrito e <em> para itálico

podemos criar também um texto marcado usando a tag <mark>

texto grande e pequeno

estamos criando um texto grande usando a tag <big> e um pequeno usando a tag <small> nesse parágrafo.

texto deletado

podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

texto inserido

podemos marcar um texto como inserido para dar uma ênfase e indicar que foi adicionado depois.

Existe também o sublinhado com a tag <u> (não tem semântica)

texto sobrescrito

para inserir coisa do tipo x^2+3 usando a tag <sup> fica: x2+3;

texto subscrito

para inserir coisa do tipo H2O usando a tag <sub> fica H2O;
